## 🚀 Overview

This repository provides a Gazebo-based forklift simulation environment equipped with LiDAR and two wide-angle cameras.

The main purpose of this project is to generate **ERP images** from two wide-angle camera views and simulate a warehouse-like environment for autonomous forklift perception research.

---

## 📌 Key Features

- **Gazebo-based forklift simulation**
- **Warehouse simulation environment**
- **LiDAR sensor integration**
- **Two wide-angle camera setup**
- **ERP image generation from dual wide-angle cameras**
- **ROS launch-based execution**

## 🔧 Requirements

This project is intended to run in a ROS + Gazebo environment.

Recommended environment:

- Ubuntu 20.04
- ROS Noetic
- Gazebo
- CMake
- C++

---

## ▶️ How to Run

Launch the warehouse simulation environment:

```bash
roslaunch warehouse_simulation warehouse_simulation.launch
```

---

## 📷 Sensor Configuration

The simulation includes the following sensor setup:

| Sensor | Description |
|---|---|
| LiDAR | Used for 3D spatial perception |
| Wide-Angle Camera 1 | Captures one side of the surrounding scene |
| Wide-Angle Camera 2 | Captures the opposite side of the surrounding scene |
| ERP Image | Generated by combining two wide-angle camera views |

---

## 🧭 Application

This simulation can be used for:

- Autonomous forklift perception
- Warehouse robot simulation
- LiDAR-camera sensor setup validation
- Wide-angle camera-based surrounding perception
- ERP image generation for 360-degree perception
